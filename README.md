1.🎵 音樂庫 (Music Library)
  一個簡潔、直觀的網頁開發專案，旨在向使用者介紹不同的音樂種類及其背景文化。透過互動式的摺疊選單，使用者可以輕鬆探索藍調、搖滾與饒舌等音樂風格。

2.🚀 功能特點
  -互動式探索：使用 HTML5 <details> 與 <summary> 標籤，實現無需 JavaScript 的流暢摺疊選單效果。
  -深度的音樂知識：提供專業的音樂風格介紹，包含藍調 (Blues)、搖滾 (Rock) 及饒舌 (Rap) 的起源與特色。
  -響應式佈局：內容區域自動置中，並設定最大寬度（800px），確保在不同螢幕尺寸上都有良好的閱讀體驗。
  -導覽選單：頂部整合了導覽列，預留「主頁」、「更多」與「關於我們」的連結介面。

3.🛠️ 使用技術
HTML5: 
  結構化網頁內容與語義化標籤。
CSS3:
  外部樣式表 (style.css) 進行進階美化。
  內聯樣式 (Inline Styles) 處理陰影 (box-shadow) 與圓角 (border-radius)。
