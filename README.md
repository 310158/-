🎵 音樂庫 (Music Library)一個簡潔、直觀的網頁開發專案，旨在向使用者介紹不同的音樂種類及其背景文化。透過互動式的摺疊選單，使用者可以輕鬆探索藍調、搖滾與饒舌等音樂風格。

🚀 功能特點互動式探索：使用 HTML5 <details> 與 <summary> 標籤，實現無需 JavaScript 的流暢摺疊選單效果。深度的音樂知識：提供專業的音樂風格介紹，包含藍調 (Blues)、搖滾 (Rock) 及饒舌 (Rap) 的起源與特色。響應式佈局：內容區域自動置中，並設定最大寬度（800px），確保在不同螢幕尺寸上都有良好的閱讀體驗。導覽選單：頂部整合了導覽列，預留「主頁」、「更多」與「關於我們」的連結介面。

🛠️ 使用技術HTML5: 結構化網頁內容與語義化標籤。CSS3:外部樣式表 (style.css) 進行進階美化。內聯樣式 (Inline Styles) 處理陰影 (box-shadow) 與圓角 (border-radius)。

📂 檔案結構Plaintext.
├── index.html    # 主要網頁檔案
└── style.css     # 樣式定義檔 (需確保此檔案存在以呈現完整效果)

📝 內容概覽網站目前涵蓋以下音樂類別：音樂種類核心描述藍調 (Blues)萬樂之母，根植於非裔美國人文化，以 12 小節和弦為特色。搖滾 (Rock)強調電吉他與自由叛逆精神，影響力遍及全球。饒舌 (Rap)嘻哈文化的核心，透過節奏與押韻講述社會議題與生活。

⚙️ 如何執行複製此專案到你的本地電腦。確保 index.html 與 style.css 放在同一個資料夾。使用瀏覽器（如 Chrome, Edge 或 Firefox）直接開啟 index.html 即可預覽。

💡 後續優化建議加入圖片：在每個音樂類別的詳細介紹中加入代表性樂器或藝人的圖片。音樂播放器：整合 Spotify 或 YouTube 的內嵌播放器，讓使用者直接試聽推薦歌曲。動態效果：在 style.css 中加入 transition 讓摺疊選單開合時更滑順。
